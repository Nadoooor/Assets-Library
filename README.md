# Assets-Library
My VR worlds asstes i had used with all my scripts and textures and 3D models.

-----------------------------------------
## How To Play: 
1. For now it is just for laptop and PCs or phones with a keyboard and mouse to move with the WASD. (Working on the controller moving)
2. Open the github hosted site from [here](https://nadoooor.github.io/Assets-Library) and enjoy. :heart: 

--------------------------------------
## Controls:
- WASD to move.
- T to fly (just if you want).
- left click to interact or touch the screen on mobiles. 

--------------------------------

## How to build: 
1. Clone the Repository.
2. Open the folder in VScode or any code editor. 
3. run the site with any live server plugin.
4. edit, make, fix what ever you want.
5. don't forget to give it a star.
